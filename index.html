Bem-vindo ao Via Verde

Via Verde é um projeto que visa facilitar a conexão entre corporativas de reciclagem com cidadãos e entidades carentes que precisam se livrar de resíduos que poderiam ser utilizados por essas corporações, transformando esse processo demorado em uma experiência rápida e prática.

Login Cadastro

Esta aplicação foi utilizado como Projeto Integrador na Universidade Vila Velha (UVV) no Curso Ciência da Computação das seguintes materias:


Construção de Software para Web

Design e Desenvolvimento de Banco de Dados I

Experiência e Interface com o Usuário
